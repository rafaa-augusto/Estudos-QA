# 🚀 Jornada QA: Do Zero ao Primeiro Emprego

Bem-vindo ao meu repositório de estudos em Quality Assurance! Este espaço foi criado para documentar minha evolução na área, registrando desde meus primeiros passos até o domínio de técnicas mais avançadas de teste.

## 📌 Objetivo
O objetivo deste projeto é demonstrar meu aprendizado contínuo, minha capacidade de receber feedbacks e como aplico as melhores práticas de teste de software.

---

## 🛠️ Projeto Alvo
Para os meus estudos iniciais, utilizei o site [Sauce Demo](https://www.saucedemo.com/), uma plataforma padrão para práticas de testes manuais e automatizados.

---

## 📈 Diário de Bordo e Evolução

### 📅 Etapa 1: Primeiros Casos de Teste (v1)
Nesta fase, foquei em entender a lógica do sistema e identificar cenários básicos de login.
- **O que foi feito:** Criação de 4 casos de teste iniciais.
- **Aprendizado:** Entendi que o QA precisa pensar tanto no "Caminho Feliz" quanto nos cenários de erro.
- **Arquivos:** [Acesse a versão 1 aqui](./v1-casos-iniciais)

### 📅 Etapa 2: Refinação e Boas Práticas (v2)
Após receber feedbacks sobre padrões da indústria, refinei a escrita dos testes.
- **O que mudei:** - Adotei o uso de tabelas para melhor leitura.
  - Especifiquei dados de teste (usuários reais).
  - Tornei os "Resultados Esperados" mais técnicos e visuais.
- **Novos Casos:** Adicionei o teste de campos vazios.
- **Arquivos:** [Acesse a versão 2 aqui](./v2-casos-refinados.md)
